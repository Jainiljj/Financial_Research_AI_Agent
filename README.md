## 🎯 Overview

FinSage AI is a intelligent financial research assistant that helps users analyze stock markets, track investments, and make informed decisions using real-time data and AI-powered insights. Built specifically with Indian market support, it combines multiple data sources to provide comprehensive financial analysis.

### ✨ Key Features

- **📊 Real-time Stock Analysis**: Live NSE/BSE stock data with technical indicators
- **📰 News Sentiment Analysis**: AI-powered market sentiment from financial news
- **💼 Portfolio Tracker**: Monitor your investments with performance metrics
- **💰 Personal Finance Tools**: SIP calculator, tax estimators, goal planning
- **📈 Interactive Charts**: Visualize stock performance with technical indicators
- **🔍 Stock Comparison**: Compare multiple stocks side-by-side
